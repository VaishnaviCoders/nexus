# Shiksha Cloud - School Management System

## Core Identity
Comprehensive cloud-based school management platform for Indian educational institutions
Connects students, parents, teachers, and administrators on unified platform
Built specifically for Indian education system requirements

## Technical Stack
Frontend: Next.js 15 with App Router, TypeScript
Backend: Next.js API Routes, Server Actions
Database: PostgreSQL with Prisma ORM
Auth: Clerk (secure, passwordless, multi-factor)
Storage: Supabase
UI: ShadCN UI components, Tailwind CSS
Payments: PhonePe integration
PWA: Progressive Web App capabilities
